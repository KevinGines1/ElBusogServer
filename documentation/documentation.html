POST: /api/addFoodPlace

This should only be used when a user with a Business_owner account will add their own food place

https://ancient-garden-70007.herokuapp.com/api/addFoodPlace

Parameter:

(
foodPlaceName: string, 
priceRange: string ("<60", "60-100", ">100"),
description : string (up to 280 chars only),
openTime: string in 24h format (e.g. "900"), NULL if 24 hours open,
closeTime: string in 24h format (e.g. "2100"), NULL if 24 hours open,
daysOpen: string (e.g. "Monday-Sunday"),
owner: string (username of place owner)
)

Return values: 

					
{
 "msg": "Succesfully added new food place!"
}

POST: /api/register

This should only be used when username and email are validated/unique

https://ancient-garden-70007.herokuapp.com/api/register

Parameter:

(
name: string (up to 30 chars), 
username: string (should be unique, up to 15 chars),
email : string (should be unique, up to 25 chars),
password: string,
picture: string (e.g. should be a link, place this in src; if none is passed, place NULL),
type: string (either: "Customer" or "Business_owner" CASE SENSITIVE)
)

Return values: 

					
{
 "msg" : "Successfully added user!"
}
