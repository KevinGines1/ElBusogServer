POST: /api/addFoodPlace

This should only be used when a user with a Business_owner account will add their own food place

https://ancient-garden-70007.herokuapp.com/api/addFoodPlace

Parameter:

(
foodPlaceName: string,
location: string, (should only be: "within UPLB", "Grove", "Raymundo", "Demarces", You can place multiple in one string e.g. "Raymundo, Demarces")
priceRange: string ("<60", "60-100", ">100"),
description : string (up to 280 chars only),
openTime: string in 24h format (e.g. "900"), place NULL if 24 hours open, VALID VALUES ARE FROM 100 TO 2400,
closeTime: string in 24h format (e.g. "2100"), place NULL if 24 hours open VALID VALUES ARE FROM 100 TO 2400,
daysOpen: string (e.g. "0123456" where 0=Sunday, 1=Monday, ..., 6=Saturday),
foodTypes: string (e.g. "Meat" or "Meat, Vegetable, Snacks, Ice Cream"; Place all in a SINGLE string separated by commas)
owner: string (username of place owner)
)

Return values: 

					
{
 "msg": "Succesfully added new food place!"
}
